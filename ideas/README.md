## Design north star (from HANDOFF)
Each animal is a different *relationship*, not a reskin. Keep it kind, warm, a little funny.
Petting/feeding/cleaning = cozy & generous; the skill test lives elsewhere or in the volume/timing.
Short, self-terminating, instant. Every level ends with a character's graded reaction
(mom bird / zookeeper / farmer / volunteer).
